# Python-Commands

## 1. Grade Checker

In this program, a user enters marks or score as input, and the program displays the corresponding grade based on predefined conditions. Conditional statements (if, elif, and else) were used to compare the entered marks with grade ranges such as A, B, C, D, and F. This task helped in understanding decision-making and conditional execution in Python programming.

## 2. Student Grades Dictionary

In this task, a dictionary data structure was used to store student names as keys and their grades as values. The program allows users to add new student records, update existing grades, and display all stored student data. Basic dictionary operations along with conditional statements were used to check whether a student already exists before updating or adding records. This task demonstrated data storage, retrieval, and modification using dictionaries in Python.

## 3. Write to a File

This program focused on file handling in Python. A text file was created, and content was written into it using file handling functions. The open() function was used in write mode, and the write() function was used to store text inside the file. This task helped in understanding how Python can create and manage files programmatically.

## 4. Read from a File

In this task, content stored in a text file was read and displayed on the screen. The file was opened in read mode using the open() function, and the read() function was used to retrieve file content. This task demonstrated how Python can access and display stored information from files.

Conclusion

Through this assignment, important Python programming concepts such as conditional statements, dictionaries, and file handling operations were practiced. These tasks helped in understanding how Python manages decision-making, data storage, file creation, and file reading operations, which are fundamental concepts in programming and software development.
